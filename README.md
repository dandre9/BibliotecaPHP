# Sistema de Biblioteca em PHP

O projeto é um sistema simples de biblioteca, onde o usuário entra através de um login e senha, procura por livros no banco de dados e, caso o livro estiver disponível, pega ele emprestado.
O usuário sabe que o livro esta disponível ou não pois a tabela que lista os livros da biblioteca indica com cores os livros emprestados (vermelho) e os não emprestados (COR?).
